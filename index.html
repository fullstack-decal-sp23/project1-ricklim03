Awakening of Azathoth

Awakening of Azathoth was my first Java project and it required me to create guessers and choosers. These guessers would take in the users guesses, create a map of the guesses, and return the next "right" guess. The choosers take in the word length and a file of words and then returns a word from the files of the matching word length to be the new secret word.

Linked List Deque

Created a doubly linked list deque with two sentinel nodes and pointers. I created two pointers, front and back, so that the front of the last element links back to the sentinel or the front of the list and the back of the first element links back to the sentinel.
